Функциональные характеристики ПО

Документация, содержащая описание функциональных характеристик программного обеспечения

1. Введение

Данный документ описывает функциональные возможности и архитектурные особенности программного обеспечения "Физминутка" (далее — Программа или ПО).

2. Общее описание программы 

Программа для ЭВМ "Физминутка" представляет собой мобильное приложение для сотрудников организаций, направленное на повышение их благополучия и улучшение качества жизни через разнообразные активности и программы. Программа предоставляет пользователю удобные инструменты для отслеживания физического и ментального здоровья. Программа реализована в виде мобильного приложения с персонализированными настройками и контентом для каждой организации.

Для пользователей ПО предоставлены следующие функциональные возможности:
Офисная и производственная гимнастика: возможность выполнять физические упражнения за рабочим местом.
Прогресс: возможность установить и отслеживать прогресс по целям, связанным с физической активностью, ментальным здоровьем и профессиональным развитием.
Тестирование: возможность проходить тесты, например по правильному обустройству рабочего пространства или по подбору подходящего вида спорта.
События: Создание групповых событий для участников программ.

3. Пользовательский интерфейс программы

Пользовательский интерфейс Программы создан с учетом удобства быстрого перехода к основным разделам приложения и включает в себя следующие компоненты:

  • Главный экран
  • Программы
  • События
  • Тестирование
  • Сторисы
  • Рейтинг
  • Панель управления
  • Настройки

Описание компонентов ПО

Главный экран

Главный экран приложения представляет обзор основных активностей:

  • Программы
  • Сторисы
  • Статистика по видам активности.

Выбор программы

  • Подбор групп упражнений для каждого сотрудника.
  • Не требует тренажеров и турников.
  • Можно выполнять прямо за рабочим местом, даже в строгом костюме или миниюбке.

События и группы

  • Сотрудники объединяются в клубы по интересам.
  • Календарь событий отражает мероприятия, которые проводятся в вашей компании.

Тестирование

  • Сотрудники могут проходить тестирование и получать рекомендации по их завершению.

Полезные сторисы

  • Актуальные сторисы на тему здоровья выходят ежедневно
  • Можно просматривать их нажав на соответствующий раздел
  • Разнообразные темы: офис, стресс, выгорание, юмор, общение и т.п.

Рейтинг и поощрение

  • Сотрудники за активности в приложении получают баллы и соревнуются между собой.
  • Победителям месяца в Панели управления автоматически формируются красочные именные дипломы.

Панель управления

  • Для контроля и анализа эффективности
  • Добавление новых клубов по интересам, коррекция информации о подарках для лидеров месяца и т.д.

Настройки

  • Просмотр и изменение профиля (сменить пароль, удалить аккаунт)
  • Настройка получения уведомлений

Технические характеристики

Программы для ЭВМ "Физминутка" реализовано на платформе iOS и Android. Для удобства пользователей предусмотрены автоматическая синхронизация активности и отображение данных в реальном времени.
